# BrightMinds Tutoring Website

A responsive tutoring website created for BrightMinds Tutoring. The website provides information about our tutors, subjects offered, grades taught, pricing, location, schedule, and contact details.

## Features

* Professional landing page
* Responsive navigation menu
* Hero section
* About BrightMinds section
* Services section
* Subjects offered
* Grades taught
* Tutor profiles with photos
* Pricing information
* Location and schedule information
* Contact section
* Separate contact form page
* Mobile-friendly design using CSS Flexbox and Media Queries

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries
* Git
* GitHub

## Project Structure

* `index.html` – Main website page
* `contact.html` – Contact form page
* `styles.css` – Main website styles
* `contact_styles.css` – Contact form styles
* `images/` – Logo and tutor images

## Location

BrightMinds Tutoring operates from Kagisanong Creche.

## Schedule

* Monday to Friday
* 4:00 PM – 6:00 PM

## Pricing

Tuition is available for R200 per month.
